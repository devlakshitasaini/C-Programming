# C Programming Practice Collection

This repository contains a collection of C programs created for learning and practicing fundamental programming concepts. The programs are organized by topic so that it becomes easier to understand different concepts step by step.

The main objective of this repository is to improve problem-solving skills and gain a strong understanding of the C programming language through consistent practice.

---

## Repository Layout

```
C-Programming
│
├── C_Basic
│   Basic introductory programs in C
│
├── If_Else
│   Programs demonstrating conditional statements
│
├── Loops
│   Programs based on different looping structures
```

---

## Concepts Practiced

Some of the important concepts covered in this repository include:

* Basic structure of C programs
* Taking input from users and displaying output
* Decision making using conditional statements
* Repetition using loops
* Simple logical and number-based problems

These programs help in understanding how programming logic works in real problems.

---

## Development Environment

The programs in this repository are written and tested using:

* **Language:** C
* **Compiler:** GCC (MinGW)
* **Code Editor:** Visual Studio Code

---

## Learning Goal

The purpose of maintaining this repository is to:

* Practice C programming regularly
* Build logical thinking ability
* Maintain a structured collection of practice programs
